**Shifra AI** lets you create a fully customizable, voice-enabled AI assistant and embed it on **any website** with a single `<script>` tag. Powered by Google Gemini, it answers visitor questions, navigates pages via voice commands, and delivers a premium conversational experience вҖ” all without writing a single line of AI code.

## рҹҡҖ Quick Start

### Prerequisites

- **Node.js** v18+ and **npm**
- **MongoDB** (local or [MongoDB Atlas](https://www.mongodb.com/atlas))
- **Firebase** project with Google Auth enabled
- **Google Gemini API Key** вҖ” [Get one here](https://aistudio.google.com/app/apikey)
- **Razorpay** account (for billing) вҖ” [Sign up here](https://razorpay.com/)

### 1. Clone the Repository

```bash
git clone https://github.com/Jenil-Mungalpara/Shifra_AI.git
cd Shifra_AI
```

### 2. Setup the Server

```bash
cd Server
npm install
```

Create a `.env` file in the `Server/` directory:

```env
PORT=8000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
RAZORPAY_KEY_ID=your_razorpay_key_id
RAZORPAY_KEY_SECRET=your_razorpay_key_secret
GEMINI_API_KEY=your_gemini_api_key
```

Start the server:

```bash
npm run dev
```

### 3. Setup the Client

```bash
cd Client
npm install
```

Create a `.env` file in the `Client/` directory:

```env
VITE_FIREBASE_API_KEY=your_firebase_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
VITE_RAZORPAY_KEY_ID=your_razorpay_key_id
```

Start the client:

```bash
npm run dev
```

### 4. Open in Browser

Navigate to `http://localhost:5173` вҖ” you're all set! рҹҺү

1. **Embed** вҖ” Paste the `<script>` tag before `</body>` on your website
2. **Speak** вҖ” Users click the mic button and speak their question
3. **Process** вҖ” The speech is transcribed and sent to the Shifra API
4. **AI Response** вҖ” Gemini AI generates a contextual response based on your business config
5. **Navigate** вҖ” If the user asked to open a page, the assistant redirects automatically
6. **Speak Back** вҖ” The response is spoken aloud via text-to-speech

## рҹ”Ң API Endpoints

### Authentication
| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/api/auth/google` | Google sign-in / sign-up |
| `POST` | `/api/auth/logout` | Log out (clear cookie) |

### User
| Method | Endpoint | Description |
|---|---|---|
| `GET` | `/api/user/current-user` | Get current authenticated user |
| `POST` | `/api/user/save-assistant` | Save assistant configuration |

### Assistant (Public)
| Method | Endpoint | Description |
|---|---|---|
| `GET` | `/api/assistant/config/:userId` | Get assistant config for widget |
| `POST` | `/api/assistant/ask` | Send a message to the AI assistant |

### Billing
| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/api/billing/order` | Create a Razorpay order |
| `POST` | `/api/billing/verify` | Verify payment signature |
